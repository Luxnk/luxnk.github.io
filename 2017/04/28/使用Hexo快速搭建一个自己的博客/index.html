使用Hexo快速搭建一个自己的博客

发表于 2017-04-28

环境要求

  • Git
  • Node.js

请先下载并安装这两项环境，这里不做过多叙述。

安装Hexo

打开Node.js命令提示符。

cd到你要防止站点的工作目录。

执行指令安装Hexo：

npm install hexo-cli -g

npm insatll hexo --save

检查Hexo是否安装完毕：

hexo -v

如果成功显示出Hexo的版本号，那么说明Hexo已经安装成功了。

初始化站点

接着上面的操作完成后，执行下面指令进行初始化：

hexo init

然后输入：

npm install

这之后默认的博客站点就已经创建完成了，接下来是看看它长什么样。

首先生成静态页面：

hexo g

然后启动内置服务器：

hexo s

服务器默认端口号为4000，如果不出意外的话，打开浏览器输入http://localhost:4000应该就能看到效果了。

要关闭服务器只需按Ctrl+C即可。

发布文章

可以使用以下指令新建文章：

hexo new 文章名

新建的文章为Markdown类型，存放在\source\_posts里面。

也可以直接在该文件夹下新建一个.md文件进行编辑。

更换主题

主题保存在\themes\文件夹下。

更换主题，在_config.yml文件中找到theme一项进行配置，这里要与\themes\文件夹下的主题所在文件夹名一致。

下面这个页面列出了Hexo现有开源主题的地址：

https://github.com/hexojs/hexo/wiki/Themes
